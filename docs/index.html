Tentang Covid Dodge

Gambaran Umum

Covid Dodge adalah game 2D yang dapat diakses untuk platform desktop. Isi dari game ini adalah pemuda yang berlari di luar dan menghindari virus covid yang tersebar. Virus covid digambarkan dengan bentuk bola berduri. untuk menghindari virus virus tersebut pemuda terbang menggunakan jetpack. Pemuda memiliki tiga nyawa dan setiap terkena virus maka nyawa berkurang satu. Game akan berakhir ketika nyawa habis.

Konsep Game

Covid Dodge terinspirasi dari game Jetpack Joyride yang dikombinasikan dengan pandemi yang sedang berlangsung. Game ini memiliki misi untuk menghindari rintangan berupa virus dan mencetak score sebanyak-banyaknya. Untuk memainkannya, pengguna hanya perlu menekan spasi pada keyboard. Karakter diberi tiga nyawa yang disimbolkan dengan hati. JIka karakter mengenai virus maka nyawa akan berkurang satu. Setelah nyawa yang dimiliki karakter habis, maka game akan berakhir dan menampilkan score yang didapatkan selama bermain.

Proses Pengembangan Game

1

Menentukan Konsep Game

Pada tahapan ini, tim kami menentukan terkait dengan konsep game seperti bagaimana game dapat dimainkan, karakter game, hingga target dalam game ini.

2

Membuat Desain Rancangan Game

Setelah menentukan konsep awal dari game, selanjutnya membuat rancangan game. Pada tahapan ini membuat desain tampilan game dan berbagai macam komponen yang ada dalam game seperti desain karakter, background, rintangan, dan objek-objek lainnya.

3

Pengkodean Game

Dalam tahapan ini, anggota tim mulai melakukan coding dan mengembangkan game dengan menggunakan Unity Engine.

4

Finishing dan Build

Setelah selesai melakukan coding, selanjutnya masuk tahap Finishing . Pada tahap ini yang dilakukan adalah merapikan kembali kode dan tampilan game. Kemudian setelah selesai dilakukan build untuk menghasilkan file game yang telah jadi.
